# PeeringLLM-Bench

PeeringLLM-Bench is the public repository for the code and datasets used in the paper:

PeeringLLM-Bench: Evaluating Large Language Models for BGP Configuration Tasks.

This repository contains the scripts, benchmarks, and evaluation framework developed to assess the capability of LLMs in generating and interpreting Border Gateway Protocol (BGP) configurations.

## Contents

Benchmark dataset of BGP configuration tasks
Evaluation scripts and scoring framework
Reference configurations and prompts
